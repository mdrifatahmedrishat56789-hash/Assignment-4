1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Answer : getElementById : Finds an element using its id
         getElementByClassName : Finds an element using class Name
         querySelector :  Find First Match CSS
         querySelectorAll : Find All Matches CSS

3. What is Event Bubbling? And how does it work?
Answer : Event Bubbling is a concept in JavaScript where an event starts from the target element and then bubbles up  to its parent elements.
Child to Parent to GrandParent to Document


5. What is the difference between preventDefault() and stopPropagation() methods?
Answer:  preventDefault() :
It stops the default behavior of an element

stopPropagation() :
It stops the event from bubbling up
